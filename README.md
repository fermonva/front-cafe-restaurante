## Ejecutar en desarrollo

### 1. Clone el repositorio:

```
git clone https://github.com/fermonva/front-cafe-restaurante.git
```

### 2. Ingresar en el directorio:

```
pnpm install
```

### 3. Ejecutar composer para instalar las dependencias:

```
pnpm run dev
```
